variance
  smaple s^2 =  SUMx=1^n (xi - _x)^2 / n-1
  population σ^2 =  SUMx=1^N (xi - μ)^2 / N

stabdard deviation `more useful`
  sample s =  odmocnina(SUMx=1^n (xi - _x)^2 / n-1)
  population σ =  odmocnina(SUMx=1^N (xi - μ)^2 / N)

coefficent of variation Cv

covariance
  sample Sxy = SUMx=1^n (xi - _x) * (yi - _y) / n-1
  population σxy =  SUMx=1^N (xi - μx)* (yi - μy) / N

linear correlation coefficent
  cov(x,y) / stdev(x) * stdev(y) = cov(y,x) / stdev(y) * stdev(x) `correlation is symetric`
  0 = independent
  -1 = perfect negative correlation
  -1 to 0 = imperfect negative correlation
